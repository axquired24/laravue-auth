# Laravel - Vue Authentication Sample

### How To
- composer install
- copy .env.example to .env
- php artisan key:generate
- php artisan migrate
- composer install

### Run Apps
- npm run watch
- (on new tab) php artisan serve
- Point browser to http://localhost:8000
